# Huzaifa Shaikh | Cybersecurity Portfolio
Cybersecurity enthusiast portfolio inspired by modern security operations centers. Showcasing certifications, security projects, threat analysis, and technical capabilities through an interactive and responsive experience.

A modern cybersecurity portfolio built with React, Vite, and Framer Motion.

## Features

- Interactive Cybersecurity-Themed UI
- Security Certifications Showcase
- Security Projects Dashboard
- Responsive Design
- Smooth Animations
- Resume Download
- Contact Information

## Tech Stack

- React
- Vite
- Framer Motion
- CSS3
- React Icons
